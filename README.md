# Vector Math Library

<h2>Introduction</h2>
A vector maths library written in Java for 2D or 3D projects.

Created by Jaspreet Dhanjan.

<h2>Features</h2>
- 4x4 Matrix type
- 2D and 3D Vectors
- Utilities like orthographic and perspective matrices
- Well tested
- No external libraries needed
